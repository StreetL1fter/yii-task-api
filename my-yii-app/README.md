# Task API (Yii2)

REST API для управления задачами.  
Реализованы все CRUD-операции через `TaskController` с вынесенной бизнес-логикой в `TaskService`.

## Endpoints

- `POST /tasks` — создать задачу  
- `PUT /tasks/{id}` — обновить задачу  
- `DELETE /tasks/{id}` — удалить задачу  
- `GET /tasks/{id}` — получить задачу  

## Особенности
- Валидация входных данных
- Обработка ошибок (404, 422, 200, 201, 204)
- Сервисный слой для изоляции бизнес-логики

## Как запустить
cd my-yii-app
php yii serve
